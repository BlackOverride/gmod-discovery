# gmod-discovery

Enhances the Garry’s Mod server browser by modifying the menu Lua file to display invisible servers.

## Installation

1. Navigate to your Garry’s Mod installation directory:

   ```
   garrysmod/lua/menu/
   ```
2. Rename your original `mainmenu.lua` to:

   ```
   mainmenu.lua.bk
   ```
3. Place the `mainmenu.lua` file from this repository into the same directory.

4. Enjoy freedom of speech
